# Images Directory

This directory contains placeholder images for the portfolio website. When actual images are available, they should be placed in this directory with the following filenames:

- `profile.jpg` - Professional headshot
- `trick-gs-results.jpg` - Trick-gs 3D point clouds and novel views
- `cheapnvs-paper.jpg` - CheapNVS paper header and architecture diagram
- `lp-ioanet-results.jpg` - LP-IOANET document shadow removal results
- `lp-ioanet-comparison.jpg` - LP-IOANET document shadow removal visual comparison
- `mobilevos-results.jpg` - MobileVOS video segmentation results
- `mobilevos-architecture.jpg` - MobileVOS architecture diagram
- `federated-learning-diagram.jpg` - Federated learning architecture diagram
- `patent-collage.jpg` - Patent front pages collage

The download.py script can be used to automatically download these images from the live website once it's deployed.
